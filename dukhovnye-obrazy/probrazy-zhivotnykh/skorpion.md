# Скорпион

### 🔍 Кратко: кто такой скорпион?

**Скорпион** — это:

* маленькое, скрытое животное,
* ночной охотник,
* внезапно атакует,
* жало его — яд, который вызывает боль, страх, иногда смерть.

📌 В Библии он упоминается как:

> _"животное с жалом"_\
> → **Луки 10:19; Откровение 9:3–5**

И это делает его прообразом:

> _духовного противника_,\
> &#xNAN;_&#x43A;оторый действует скрыто_,\
> &#xNAN;_&#x43D;о причиняет глубокую рану_.

***

### 📖 Основные места Писания

| Книга                | Стих                                                                | Тема                                                   |
| -------------------- | ------------------------------------------------------------------- | ------------------------------------------------------ |
| **Луки 10:19**       | _"Вот, даю вам власть наступать на змей и скорпионов…"_             | Христос даёт победу над силами тьмы                    |
| **Откровение 9:3–5** | _"Саранча имела власть… какую имеют скорпионы земли"_               | Духовные мучители, которые "жалом" причиняют страдание |
| **Бытие 3:15**       | _"Семя женщины будет поражать голову змия…"_                        | Прообраз борьбы с обольстителем                        |
| **Псалом 90:13**     | _"На змия и на скорпиона наступишь ногами своими"_                  | Победа через Господа                                   |
| **Притчи 30:28**     | _"Скорпион — ходит рукою своею"_                                    | Образ коварства и лукавства                            |
| **Иезекииль 2:6**    | _"Не страшись их, не страшись слов их… среди скорпионов живёшь ты"_ | Пророк стал между духами, которые терзают душу         |
| **Исаия 59:5**       | _"Яйца скорпиона едят..."_                                          | Ложь, которая рождает смерть                           |

📌 Скорпион — не просто насекомое,\
а символ того, что:

> _"враг может быть рядом, но невидим",_\
> &#xNAN;_"его удар — внезапен",_\
> &#xNAN;_"его яд — отравляет мысли и сердце"._

***

### 🧬 Этимология слова «скорпион» в Писании

| Слово                             | Иврит / Греческий | Значение                                                                       |
| --------------------------------- | ----------------- | ------------------------------------------------------------------------------ |
| **עַקְרַב (**_**акрав**_**)**     | Иврит             | "скорпион" → также может значить "противостояние", "зло", "тайную атаку"       |
| **σκορπίος (**_**скорпиос**_**)** | Греч.             | "скорпион" → связан с корнем **σκορπός** – "рассеивание", "разрушение"         |
| **μάστιγα (**_**мастига**_**)**   | Греч.             | "жало", "наказание" → в Откровении 9:10 — символ боли                          |
| **βάσανος (**_**басанос**_**)**   | Греч.             | "испытание", "мука", "страдание" → Откр. 9:5 — указывает на длительное мучение |
| **δέκα (**_**дека**_**)**         | Греч.             | "десять месяцев" → временной период духовного испытания                        |

📌 Слово «скорпион» говорит не только о боли,\
но и о **тайной власти**,\
и даже **инструменте суда**,\
через который Бог:

* предупреждает,
* испытывает,
* или наказывает тех, кто отвергает Его.

### 🌍 Скорпион в природе ↔ духовный враг

| Признак скорпиона             | Духовная параллель                                      |
| ----------------------------- | ------------------------------------------------------- |
| **Маленький, но опасный**     | _"Даже один соблазн может погубить многих"_             |
| **Живёт в темноте**           | _"Он — князь мира сего, который любит тьму"_            |
| **Прячется под камнями**      | _"Он притаился в вашем сердце"_                         |
| **Жало с ядом**               | _"Страх, который парализует"_                           |
| **Удар без предупреждения**   | _"Искушение приходит, когда вы не готовы"_              |
| **Яд разрушает тело**         | _"Грех разрушает душу и тело"_                          |
| **Выжидает момент для удара** | _"Диавол ходит, как лев рыкающий, ища, кого поглотить"_ |

📌 Скорпион — это не просто образ боли, а **прообраз духовного врага**, который:

> _не виден до удара_, _но его яд может оставить след на всю жизнь_.

### ⚖️ Скорпион ↔ змей — сравнение

| Признак                | Скорпион                        | Змей                              |
| ---------------------- | ------------------------------- | --------------------------------- |
| **Оружие**             | Жало с ядом                     | Шипы и обман                      |
| **Атака**              | Внезапная боль                  | Искуситель и обольститель         |
| **Символ**             | Страх, боль, паралич            | Грех, гордыня, отступление        |
| **Действие**           | Яд внутри → медленная разруха   | Удар → падение за один раз        |
| **Образ в Откровении** | _"Саранча с жалами скорпионов"_ | Откр. 9:3–5 — духовное мучение    |
| **Прообраз**           | _"Все они будут судимы"_        | _"Он будет поражать голову змия"_ |
| **Победа**             | _"Через веру и Слово"_          | _"Через кровь Агнца"_             |

📌 Скорпион — более **скрытый и терзающий**, змей — **соблазнитель и обманщик**. Но оба — прообразы **работы врага**, который хочет:

> _"внушить страх_, _отвести от пути правды_, _и сделать тебя рабом боли."_
